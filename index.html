📊 Analytics

🔒 API Key Security — How it works

Your Groq API key is stored in localStorage — that means it lives only in your browser on your device. It is never stored on any server, never in the HTML file, and never sent anywhere except directly to Groq's servers when you click Generate. Anyone who uses your site enters their own free key. This is the safest possible approach for a free hosted site.

Visit js.org → click "Get yours now" → submit a pull request with your desired name (e.g. factflow.js.org). It's free for GitHub Pages sites.
